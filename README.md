# 🏏 IPL Match Winner Prediction (2008–2025)

Machine Learning based system to **predict the winner of IPL matches** using historical match data, toss details, venue conditions, and team performance.

---

## 📌 Project Overview

This project analyzes IPL match data from **2008 to 2025** and builds ML models to predict match outcomes.

It includes:
- Data Cleaning & Preprocessing
- Feature Engineering
- Exploratory Data Analysis
- Model Training & Evaluation
- Match Winner Prediction

**Models used:**
- Random Forest
- Logistic Regression

---

## 🎯 Objectives

- Understand IPL dataset (2008–2025)
- Analyze patterns affecting match outcomes
- Build an accurate prediction model
- Evaluate model using standard ML metrics

## ⚠️ Important — Dataset Instruction (Must Read)

Due to GitHub file size limits, the dataset is provided in **compressed (ZIP/RAR) format**.

👉 **Before running the notebook, you MUST extract the dataset.**

### Steps to follow:

1. Download the compressed dataset from this repository
2. Right click → **Extract Here**
3. Ensure `IPL.csv` is placed in the **same folder** as the notebook file

If not extracted, the notebook will show a **File Not Found Error**.
